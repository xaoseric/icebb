Installation

Here's how to get IceBB up and running on your site in no time!
  1. Find a web host. If you already have one, you can skip this step. XAOS Interactive recommends A Small Orange, but any host with PHP 4.3+ or PHP 5+ should work just fine.
  2. Upload IceBB to your web host using an FTP client.
  3. Go to where you uploaded IceBB in your web browser. For example, if you uploaded IceBB to the forums folder and your site was example.com, then you'd go to http://www.example.com/forums/.
  4. IceBB's installer should guide you through the rest. If not, you can ask for help on our support forum.
